## Problem Statement

Given two positive integers a and b (where a, b < 1000), the problem asks you to calculate and return the integer value of the hypotenuse squared of a right-angled triangle. 

In mathematical terms, compute:
c^2 = a^2 + b^2

## Approach

1. Read the input values for a and b.
2. Compute the square of a and the square of b.
3. Sum the two squared values together.
4. Print the final integer result.
